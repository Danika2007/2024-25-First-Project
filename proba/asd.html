Bubble Sort Rendezési Algoritmus

Algoritmus Leírása

A Bubble Sort egy egyszerű rendezési algoritmus, amely a legnagyobb (vagy legkisebb) elemet "buborékoltatja" a sorozat végére (vagy elejére). Az algoritmus többször végigiterál a listán, és minden iterációban összehasonlítja az egymás melletti elemeket, és szükség esetén cseréli őket. Ez addig folytatódik, amíg az egész lista rendezetté nem válik.

Hatékonyság és Tárigény

Időbeli hatékonyság: A Bubble Sort algoritmus legrosszabb és átlagos időbeli hatékonysága O(n2), ahol n a lista elemszáma. A legjobb esetben, ha a lista már rendezett, az algoritmus O(n) időbeli hatékonysággal rendelkezik.

Tárigény: Az algoritmus helyigénye O(1), mivel csak egy néhány segédváltozóra van szükség.

Kód Példa

Az alábbi példa a Bubble Sort algoritmust mutatja be Python nyelven:

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
